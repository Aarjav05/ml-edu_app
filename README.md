Overview:-
>An interactive machine learning visualization tool built with Flutter.
>Users can place and manipulate data points on a canvas while a neural network trains in real time, allowing the decision boundary to update instantly.

Key Features:-
>Interactive dataset editing (add, move, or remove points)
\n
>Real-time neural network training
>Live visualization of decision boundaries
>Multiple synthetic datasets (moons, circles, XOR, spiral, Gaussian)
>Adjustable network size and activation functions
>Learning rate control
>Mobile-friendly UI with responsive layout
>Background training using Dart isolates


ML Engine:-

>The neural network is implemented in pure Dart as a small multi-layer perceptron.

>Default architecture:-
2 → 16 → 16 → 1

Overview:-
_An interactive machine learning visualization tool built with Flutter.
_Users can place and manipulate data points on a canvas while a neural network trains in real time, allowing the decision boundary to update instantly.

Key Features:-
_Interactive dataset editing (add, move, or remove points)

_Real-time neural network training
_Live visualization of decision boundaries
_Multiple synthetic datasets (moons, circles, XOR, spiral, Gaussian)
_Adjustable network size and activation functions
_Learning rate control
_Mobile-friendly UI with responsive layout
_Background training using Dart isolates


ML Engine:-

-The neural network is implemented in pure Dart as a small multi-layer perceptron.

-Default architecture:-
2 → 16 → 16 → 1
